Privacy & Ethics Statement (Prototype)

PancrePal is an educational prototype for personal awareness. It does not provide medical advice or replace clinical guidance. All data is stored locally in a private PostgreSQL database on this computer. No data is shared with third parties.

Data Collected

Glucose values, optional meal names, mood emojis, optional time-of-day labels, and timestamps.

User Control

Users can create, update, and delete entries at any time. There is no synchronisation or external transfer.

Limitations

This is a student project. It is not a medical device and should not be used to make clinical decisions.
